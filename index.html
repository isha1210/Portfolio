Artifact 1 - Use of AI in semiconductor industry

Category Domain Type
Introduction

A small project to showcase how automating can be used in the semiconductor industry.

Description

This artifact is based onthe research part of the AI lab using STORM.ai. There are many EDA tools that are used in the industry to automate flows and get desired results. This is one of them.

Objective

The objective of this artifact is to demonstrate how automation can be used to analyze Static Timing Analysis (STA) reports in semiconductor design workflows.This project was created to explore how scripting and data processing techniques can reduce manual effort when analyzing large timing reports and identifying slack violations.

Process to reproduce
  1. Phase 1 — Open my github repository and copy the timing report. This will be your input. You can have your own file too.
  2. Phase 2 — Make sure you have a python editor on your system. If not, you can use google colab. I used Pycharm
  3. Phase 3 — You can copy my py code from github and run the file in py editor
Tools & Technologies
Python for parsing and analyzing timing report data TCL commonly used scripting language in EDA tools GitHub to host the project and documentation Pandas and Regex libraries – for text parsing and data analysis ChatGPT for generating a sample STA report
Value Proposition
  • This artifact demonstrates my ability to apply automation techniques to semiconductor design workflows. Parsing timing reports manually can be time-consuming, especially for large designs.
  • Automating this process helps engineers quickly identify critical paths and slack violations

AI/ML Timelines

Category Domain Type
Introduction

Humans have always developed tools to solve problems and improve productivity. From early inventions like fire and the wheel to modern computers and the internet, technology has continuously evolved. Today, artificial intelligence (AI) represents a major technological advancement where machines can perform tasks that usually require human intelligence.

Description

Artificial intelligence began developing in the 1950s when Alan Turing introduced the idea that machines could think. His proposed Turing Test became a foundation for AI research. The field formally started in 1956 at the Dartmouth Conference. Later milestones include expert systems in the 1980s and IBM Deep Blue defeating a chess champion in 1997. Progress increased in the 2000s due to better computing power and data availability.

Objective

The objective is to study the history of AI, understand the causes of AI winters, and explore modern advancements that have led to AI’s widespread adoption.

Process
  1. Phase 1 — This study involves reviewing historical AI developments, analyzing funding fluctuations, and examining modern AI applications and technological contributions.
  2. Phase 2 — AI winters were periods when research funding and interest declined. The first AI winter in the 1970s occurred due to limited computing resources and unrealistic expectations. A second decline in the late 1980s happened when expert systems failed commercially. These challenges slowed innovation until improvements in hardware and data revived AI in the early 2000s.
  3. Phase 3 — Recent AI growth is driven by deep learning, big data, and cloud computing. AI is now widely used in healthcare, finance, robotics, and transportation. Applications include virtual assistants, surgical robots, and self-driving cars. Companies like IBM, Tesla, Intel, Apple, NVIDIA, and OpenAI have contributed significantly through innovations in AI hardware, software, and generative models.
Tools & Technologies
Key technologies include machine learning algorithms, cloud computing platforms, GPUs and CPUs, large datasets, and natural language processing systems. tool2 [Tool 3]

Unique Value

Relevance
